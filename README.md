# Proyecto-de-An-lisis-de-Censo---Excel
Análisis de datos enfocado en el dataset "Censo.xlsx". El trabajo abarca el ciclo completo de procesamiento: limpieza de datos, cálculos avanzados, modelado con Tablas Dinámicas y creación de visualizaciones profesionales mediante Dashboards interactivos. Orientado a demostrar capacidad de análisis y resolución de problemas técnicos en Excel.
